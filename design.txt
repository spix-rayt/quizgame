Админ запускает сервер java. сервер генерирует два кода, для входа на сервер как админ, и для входа на сервер как игрок.
При открытии web страницы, пользователь должен ввести своё имя и код.
По коду создаётся сессия с правами админа или с правами игрока
Ключ сессии - имя игрока (обычный ключ делать нельзя, так как нет авторизации, и при сбросе куков игрок не сможет продолжить игру)
Код сохраняется в localStorage и при перезагрузке страницы сразу используется для входа


Если папка с игрой не выбрана, то админ её выбирает и игра подгружается всем игрокам

Игрок может выбрать аватарку и имя, которые отправляются на сервер, прикрепляются к сессии и отправляются всем игрокам


При заходе на сервер, пользователю отправляется текущее состояние игры, всего их 4:
1. Админ ещё не нажал старт игры (Splash screen)
2. Показ текущей таблицы с темами и ценами вопросов
3. Показ вопроса
4. Конец игры и вывод статистики

Страницу можно перезагружать когда угодно. С помощью ключа и имени игрока будет восстанавливаться сессия.
